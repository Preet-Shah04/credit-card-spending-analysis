# Credit Card Spending Analysis

## Business Objective

Analyze customer spending behavior across cities, card types, expense categories and gender using Exploratory Data Analysis (EDA).

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Findings

- Greater Mumbai contributes the highest spending.
- Silver cards generate the highest spending.
- Bills are the dominant expense category.
- Female customers contribute higher spending than male customers.
- Travel is the least significant spending category.
